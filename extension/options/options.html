BingeBreak

BingeBreak finds the natural stopping point inside an episode — the real scene break right before the closing hook — and pauses the player there. It runs entirely on your machine: no accounts, no API keys, no LLM calls.

How the stop point is found

  1. Timestamps come from the episode's own captions — read from the player when it exposes them, or from a subtitle file (.srt/.vtt) you pick. Long dialogue silences mark real scene breaks.
  2. Plot comes from known summaries: the episode's Wikipedia article (Plot section) when one exists, otherwise the TVMaze synopsis.
  3. The summary's beats are aligned to the scenes (plot order matches screen order), locating where the closing beat starts. The stop point is the measured scene break just before it — never a fixed “N minutes before the end”.

Privacy

Network requests go only to api.tvmaze.com and en.wikipedia.org (episode lookups). Captions are analyzed locally and never uploaded. Results cache in chrome.storage.local.
